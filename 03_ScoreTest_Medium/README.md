# Español

## Indicaciones

### Dificultad: Medio

Seguimos con los ejercicios. Ahora vamos poner en práctica unos cuantos conceptos de estadística y aplicarlos. En la carpeta de este ejercicio encontrarás un archivo *.csv* el cual contiene unas calificaciones de unos alumnos, tu objetivo es presentar un gráfico comparando los grupos y realizando una prueba estadística determinar a qué grupo le fue mejor... si es que hay alguno.

Recuerda que estos ejercicios son para que tú practiques y para que aprendas nuevos conceptos o reafirmes los previos, no tengas miedo en probar nuevas herramientas.

# English

## Instructions

### Difficulty: Medium

Let's continue with the exercises. Now we are going to put into practice some statistical concepts and apply them. In the folder of this exercise you will find a *.csv* file which contains some grades of some students, your goal is to present a graph comparing the groups and performing a statistical test to determine which group did better... if there is any.

Remember that these exercises are for you to practice and to learn new concepts or reaffirm previous ones, do not be afraid to try new tests.
