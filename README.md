# ¡Hola, bienvenido! 

Me llamo Conrado Montes de Oca, en este repositorio compartiré diferentes ejercicios que puedes tratar de resolver para que puedas mejorar en R. 

Cada semana compartiré un ejercicio relacionado con estadística, con el lenguaje de R en sí mismo o algún otro ejercicio que considere importante para mejorar en este maravilloso mundo. 

La idea es que los resuelvas sin ver la solución del problema. Que los razones o que puedas interpretarlos de la mejor manera posible, cada ejercicio tendrá su explicación para que puedas comprobar si tu información es correcta. 

Cada ejercicio irá rotulado con su grado de dificultad para que puedas comenzar con los ejercicios más sencillos o si sientes que tienes el nivel, adentrarte a ejercicios más complejos. 

Si esto te sirve agradecería que compartas el repositorio, si tienes una sugerencia no dudes en contactarme y seguirme. 

Let's do some Statistics!
