# Indications

## Difficulty: Medium

### English

 Welcome to the first exercise. Let's get started.
 The challenge is to use rbase to create the following structure:

```
       *
      ***
     *****
    *******
   *********
  ***********
 *************
 ```

 Hint: Use the "cat()" function, you should read the documentation.
 Note: If you come from Python remember that R starts at 1 and is inclusive,
 while Python starts at 0 and is exclusive on the right.

 Extra: Use a variable to store the number of rows you want your pyramid to have
 to make it more dynamic.

 Good luck!
